# MigraSOS AI

MigraSOS AI is an AI-assisted hyperlocal migraine triage and care navigation system.

## Current Version

full-stack AI-integrated project setup.

## Tech Stack

- Frontend: React
- Backend: FastAPI
- AI/ML: Python, Scikit-learn
- Future Integration: Google Maps API

## Core Features

- Symptom input
- Red-flag emergency detection
- Migraine risk prediction
- Green, Yellow, Red risk level
- Nearby care navigation
